Technologies

A collection of technologies like MySQL, Express Js, Angular Js and Node Js that is used for building complex and interactive web applications. This technology stack is used to develop new modules and features.

HTML5, CSS3, Bootstrap v5.3, and JavaScript (JS)

Technologies which enables the developers to create modern, feature-rich, and user-friendly websites and web applications that work well on different devices and screen sizes. They are commonly used in combination to build responsive, interactive, and visually appealing websites with a wide range of functionality.

Angular 14

Angular is a popular open-source web application framework developed and maintained by Google, and it is used for building dynamic, single-page applications (SPAs) and progressive web applications (PWAs).

NodeJS, MySQL

Node.js provides a powerful and scalable server-side runtime environment, and MySQL can be used as a database backend to store and retrieve data. Together, they can be used to build various types of server-side applications, such as web applications, APIs, microservices, and more.

Java Standard Edition

Java SE is widely used for developing a wide range of applications, including desktop applications, enterprise applications, web applications, mobile applications, and embedded systems. Java code written on one platform can run on any platform that supports Java, making it highly portable and platform-independent.
